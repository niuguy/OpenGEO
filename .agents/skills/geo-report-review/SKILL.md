# GEO Report Review

Use this skill before sharing an OpenGEO report externally.

## Workflow

1. Check that all metrics are based on completed runs.
2. Verify the report distinguishes observation from causation.
3. Remove fake testimonials, unverifiable claims, or unsupported guarantees.
4. Check whether demo data is clearly labelled.
5. Confirm PDF/export links work in the deployed environment.
6. Summarize residual caveats.

## Output

Return:

- report readiness
- factual risks
- missing context
- recommended wording changes
