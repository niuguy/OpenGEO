# GEO Project Setup

Use this skill when setting up or reviewing an OpenGEO project.

## Workflow

1. Confirm the target business name, category, location, and website.
2. Identify known competitors and local alternatives.
3. Define target attributes that should matter in AI recommendations.
4. Check provider coverage: ChatGPT, Gemini, and optional Google AI Overview-style search data.
5. Generate or review prompt clusters for local recommendation intent.
6. Run a small sample before scheduling larger monitoring.
7. Summarize what OpenGEO can measure and what remains externally unverifiable.

## Output

Return:

- project setup status
- missing API keys or data
- prompt coverage gaps
- competitor coverage gaps
- recommended next run
